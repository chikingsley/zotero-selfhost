# References

This folder is for source inventory, upstream notes, and research breadcrumbs.

It should not become active product code. Keep runnable prototypes, harnesses, and implementation under dedicated project folders later.

## Current Local Sources

| Path | Purpose |
| --- | --- |
| `dataserver/` | Official Zotero Data Server source and tests. |
| `zotero-selfhost/` | Older Docker packaging around official Zotero server/client/web-library. |
| `zotprime/` | Older on-prem packaging ancestor. |
| `on-prem-zotero-webui/` | WebDAV PDF proxy and web-library overlay companion. |
| `sources.md` | Initial source links collected by the user. |

## Reference Note Format

When adding notes, include:

- Source URL or local path.
- Commit/date if available.
- What question the source answers.
- What is confirmed versus inferred.
- Whether it affects reference stack, compatible server, companion, or deployment.
